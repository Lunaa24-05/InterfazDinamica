Juego de atrapar flores
DescripciónL:
En el juego se mostrará una canasta café donde al moverla, el usuario tendra que atrapar las flores que vayan cayaendo de la parte superior. Cuando no logre atrapar una, el juego terminará.
Mécanica:
Al iniciar la partida, el juego comenzará y aparecerá en la pantalla de canvas una canasta con la que el usuario movera con el mouse y tendra que atrapar las flores que se vayan generando. 
Con cada flor atrapada se suman puntos.
Cuando no atrapes una flor, se termina el juego.
La velocidad de las flores irá aumentando cada 6 segundos aprox.
Tecnologias usadas:
Canvas, LocalStorage, 
asistencia Co-pilot de github
